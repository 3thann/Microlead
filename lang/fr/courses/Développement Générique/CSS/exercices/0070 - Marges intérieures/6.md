## Énoncé

Soit le code suivant : 

```html
<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Microlead - Exercice 1 CSS</title>
</head>
<body>
    <h1>Microlead</h1>
    <p>Se former sur Microlead, c'est génial !</p>
</body>
</html>
```

Modifier le code afin de :

- Définir une largeur de 600px sur le paragraphe et une marge intérieure inférieure de 90px.

## Contraintes

Les langages utilisés doivent être uniquement le HTML et le CSS.

## Résultats attendus

Le résultat suivant est visible lors du chargement de la page HTML au sein de votre navigateur :

- Le paragraphe est d’une longueur de 600 pixels et a une marge intérieure inférieure de 90 pixels.
