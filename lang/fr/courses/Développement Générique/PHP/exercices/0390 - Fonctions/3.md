## Énoncé

Soit la variable suivante :

``` php
$var1 = 39;
$var2 = 23;
```

Dans un fichier PHP, créer une fonction qui additionne les variables et retourne le résultat pour l’afficher.

## Contraintes

- Le langage utilisé doit être uniquement le PHP.
